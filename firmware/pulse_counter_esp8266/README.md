# Прошивка для ESP8266 (Wemos D1 mini)

## Описание

Код для подсчета импульсов светодиода электросчетчика с помощью датчика TEMT6000 и платы Wemos D1 mini. Подсчитанное количество импульсов отправляется в MQTT топик, указанный во входящем запросе. Все дальнейшие расчеты (перевод в кВт·ч, расчет стоимости, статистика) должны выполняться в системе умного дома (Home Assistant).

**Особенности данной прошивки:**
- Поддержка двух тарифов (день/ночь) через переключение топиков
- Регулировка порога срабатывания датчика через MQTT (команды + и -)
- Публикация статуса ESP (LWT) в топик `Counter/Available`
- Ретейн сообщений для сохранения состояния при перезагрузке
- Режим только клиента (AP отключен)

Подробнее о оригинальном проекте: [Интегрируем электросчетчик в систему УД](https://sprut.ai/article/integriruem-elektroschetchik-v-sistemu-umnogo-doma)

## Принцип работы

Светодиод счетчика мигает с частотой, пропорциональной потребляемой мощности. Передаточное число счетчика (например, 1000 imp/kWh) указывает, сколько импульсов соответствует 1 кВт·ч.

ESP8266 через датчик TEMT6000 фиксирует каждый импульс и накапливает их во внутреннем счетчике. Когда система автоматизации отправляет команду в топик `Counter/choice`, ESP публикует накопленное количество импульсов в указанный топик и обнуляет счетчик.

Таким образом, частота опроса и разделение по тарифам полностью контролируются Home Assistant, а ESP выполняет только функцию счетчика.

---

## MQTT топики

| Топик | Назначение |
|-------|-------------|
| `Counter/day` | Публикация импульсов для дневного тарифа |
| `Counter/night` | Публикация импульсов для ночного тарифа |
| `Counter/choice` | Прием команд: `day`, `night`, `+`, `-` |
| `Counter/Available` | Статус ESP: `online` / `offline` (LWT) |

---

## Команды управления

| Команда | Действие |
|---------|----------|
| `day` | Переключить публикацию в топик `Counter/day` |
| `night` | Переключить публикацию в топик `Counter/night` |
| `+` | Увеличить порог срабатывания датчика на 10 |
| `-` | Уменьшить порог срабатывания датчика на 10 |

## Схема подключения

| TEMT6000 | Wemos D1 mini |
|----------|---------------|
| VCC (средний) | 3.3V |
| GND | GND |
| OUT (крайний) | A0 |

> 💡 Датчик закрепите на светодиоде счетчика черной изолентой — это защитит от внешнего света.
