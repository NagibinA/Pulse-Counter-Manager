# Pulse Counter Manager

Интеграция для управления импульсными счетчиками электроэнергии в Home Assistant.

## Возможности

- Прямой прием импульсов от ESP через MQTT
- Автоматическое переключение тарифов по времени
- Расчет стоимости электроэнергии по тарифам
- Сохранение состояния при перезагрузке
- Поддержка нескольких счетчиков
- Отправка данных в legacy MQTT топики

## Установка через HACS

1. Добавьте репозиторий в HACS:
   - HACS → Интеграции → меню (⋮) → Custom repositories
   - URL: `https://github.com/NagibinA/Pulse-Counter-Manager`
   - Type: Integration

2. Нажмите "Скачать"

3. Перезапустите Home Assistant

## Настройка

Настройки → Устройства и интеграции → Добавить интеграцию → Pulse Counter Manager

## Параметры

| Параметр | Описание |
|----------|----------|
| Имя устройства | Название счетчика |
| Топик дня | MQTT топик для дневных импульсов |
| Топик ночи | MQTT топик для ночных импульсов |
| Дневной тариф | Стоимость кВт·ч днем |
| Ночной тариф | Стоимость кВт·ч ночью |
| Начало ночи | Время начала ночного тарифа |
| Конец ночи | Время окончания ночного тарифа |

## Сенсоры

- `sensor.{name}_total_kwh` - общее потребление
- `sensor.{name}_day_kwh` - дневное потребление
- `sensor.{name}_night_kwh` - ночное потребление
- `sensor.{name}_day_cost` - стоимость дневного тарифа
- `sensor.{name}_night_cost` - стоимость ночного тарифа
- `sensor.{name}_total_cost` - общая стоимость

## Сервисы

- `pulse_counter.set_day_kwh` - установить дневные показания
- `pulse_counter.set_night_kwh` - установить ночные показания
- `pulse_counter.reset_monthly` - сбросить месячные показания
