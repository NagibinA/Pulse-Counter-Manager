# Pulse Counter Manager

Интеграция для управления импульсными счетчиками электроэнергии, воды, газа и тепла в Home Assistant.

## Возможности

- Поддержка различных типов счетчиков (электричество, вода, газ, тепло)
- Двухтарифный учет для электроэнергии (день/ночь)
- Автоматическое переключение тарифов по времени
- Расчет стоимости электроэнергии
- Сохранение состояния при перезагрузке
- Поддержка нескольких счетчиков
- Отправка данных в legacy MQTT топики
- Сенсор "Получено импульсов" для отображения скорости поступления импульсов
- Ссылка на официальные тарифы в настройках

## Установка через HACS

1. Добавьте репозиторий в HACS:
   - HACS → Интеграции → меню (⋮) → Custom repositories
   - URL: `https://github.com/NagibinA/Pulse-Counter-Manager`
   - Type: Integration

2. Нажмите "Скачать"

3. Перезапустите Home Assistant

## Настройка

Настройки → Устройства и интеграции → Добавить интеграцию → Pulse Counter Manager

## Сенсоры для электроэнергии

- `{name} Всего` - общее потребление (кВт·ч)
- `{name} Энергия дневная` - дневное потребление (кВт·ч)
- `{name} Энергия ночная` - ночное потребление (кВт·ч)
- `{name} Энергия дневная за месяц` - потребление за месяц день (кВт·ч)
- `{name} Энергия ночная за месяц` - потребление за месяц ночь (кВт·ч)
- `{name} Энергия всего за месяц` - общее потребление за месяц (кВт·ч)
- `{name} Стоимость за дневной тариф в месяц` - стоимость дневного тарифа (руб)
- `{name} Стоимость за ночной тариф в месяц` - стоимость ночного тарифа (руб)
- `{name} Стоимость электричества за месяц` - общая стоимость (руб)
- `{name} Получено импульсов` - количество импульсов в минуту (имп/мин)

## Сервисы

- `pulse_counter.set_day_kwh` - установить дневные показания
- `pulse_counter.set_night_kwh` - установить ночные показания
- `pulse_counter.set_total_value` - установить показания однотарифного счетчика
- `pulse_counter.reset_monthly` - сбросить месячные показания
