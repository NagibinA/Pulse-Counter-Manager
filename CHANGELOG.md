# Changelog

Все значимые изменения в проекте "Pulse Counter Manager" будут описаны в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/), и проект придерживается [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.4.0] - 2024-01-15

### Добавлено

#### Интерфейс и настройки
- **Группировка настроек** — форма добавления/редактирования счетчика разбита на логические секции:
  - Для электричества: MQTT, Тарифы, Параметры, Начальные показания, Экспорт
  - Для воды/газа/тепла: MQTT, Параметры, Тариф, Начальные показания, Экспорт
- **Всплывающие подсказки** — для каждого поля конфигурации добавлено пояснение с примером
- **Предпросмотр MQTT топиков** — при вводе топика сразу показывается пример его использования
- **Улучшенная иконка и эмодзи** — в меню управления счетчиками

#### Валидация
- Проверка формата времени (ЧЧ:ММ) для полей `night_start` и `night_end`
- Проверка тарифов (значение не может быть отрицательным)
- Проверка портов MQTT (диапазон 1-65535)
- Проверка коэффициента импульсов (должен быть больше 0)

#### Документация
- Обновлен README с подробным описанием всех функций
- Добавлен CHANGELOG.md

### Исправлено

#### Асинхронность
- Исправлена асинхронная остановка опроса при выключении Home Assistant
- Метод `async_stop_polling` теперь асинхронный
- Метод `async_start_polling` теперь проверяет наличие `_update_current_tariff`

#### Обработка данных
- Добавлены отсутствующие переменные `_pending_impulses`, `_pending_day_impulses`, `_pending_night_impulses`
- Добавлен публичный метод `async_save_state()` для сохранения состояния
- Добавлены property `day_partial_impulses` и `night_partial_impulses` для доступа к накопленным импульсам

#### Сенсоры
- Исправлен доступ к защищенным атрибутам `_day_partial` и `_night_partial` через публичные property

#### Локализация
- Исправлена ошибка `MISSING_VALUE` в переводе шага `edit_tariffs`
- Добавлена передача переменной `tariff_info_url` в description_placeholders

### Изменено
- Версия интеграции обновлена до 1.4.0
- Минимальная версия Home Assistant: 2024.1.0

## [1.3.1] - 2024-01-05

### Добавлено
- Первая публичная версия
- Поддержка двухтарифных счетчиков электроэнергии
- Поддержка счетчиков воды, газа, тепла
- Экспорт показаний в отдельные MQTT топики
- Поддержка отдельного MQTT брокера для экспорта
- Автоматический расчет стоимости по тарифам
- Сохранение состояния при перезагрузке
- Сенсор "Получено импульсов" (имп/мин)
- Ссылка на официальные тарифы в настройках
