# Changelog

## [1.4.0] - 2024-01-15

### Added
- Группировка настроек в config flow (MQTT, Тарифы, Параметры, Экспорт)
- Всплывающие подсказки для всех полей конфигурации
- Предпросмотр MQTT топиков в подсказках
- Валидация формата времени (HH:MM)
- Валидация тарифов (неотрицательные значения)
- Валидация портов MQTT (1-65535)
- Валидация коэффициента импульсов (>0)

### Fixed
- Асинхронная остановка опроса при выключении HA
- Добавлены отсутствующие переменные `_pending_impulses`
- Исправлен метод `async_start_polling`
- Добавлены property для доступа к partial импульсам
- Исправлен вызов `_save_state` на `async_save_state`

## [1.3.2] - 2024-01-10

### Fixed
- Исправлена асинхронность в `__init__.py`
- Добавлены недостающие переменные в `mqtt_handler.py`

## [1.3.1] - 2024-01-05

### Added
- Первая публичная версия
- Поддержка двухтарифных счетчиков
- Экспорт показаний в MQTT
