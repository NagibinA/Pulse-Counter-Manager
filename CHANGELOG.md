# Changelog

Все значимые изменения в проекте "Pulse Counter Manager" будут описаны в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/), и проект придерживается [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.5.1] - 2024-xx-xx

### Добавлено

#### 📬 Ежемесячные уведомления о показаниях

- **Автоматическая отправка показаний** в заданный день месяца (по умолчанию 24 число)
- **Настраиваемое время отправки** (по умолчанию 19:00:00)
- **Выбор сервиса уведомлений**:
  - 📢 Уведомление в Home Assistant
  - 📱 Мобильное приложение
- **Гибкая настройка содержимого**:
  - ☀️ Дневные показания
  - 🌙 Ночные показания
  - 📈 Общие показания
  - 📅 Потребление за месяц
  - 💰 Стоимость
- **Дополнительный текст** — можно добавить напоминание или комментарий
- **Отдельный пункт меню** "Настроить уведомления" в редакторе счетчика
- **Сервис `test_notification`** для проверки настроек уведомлений

#### Новая секция в настройках счетчика

При добавлении или редактировании счетчика появилась секция **"📬 Ежемесячные уведомления"** со всеми настройками.

### Улучшено

- **Валидация** — добавлена проверка дня месяца (1-31)
- **Подсказки** — обновлены описания для полей уведомлений
- **Производительность** — оптимизирована проверка времени отправки (каждую минуту)

### Исправлено

- Корректная обработка флага отправки — уведомление приходит только один раз в месяц
- Сброс флага в первый день следующего месяца для повторной отправки

### Изменено

- Версия интеграции обновлена до 1.5.0

## [1.4.0] - 2024-01-15

### Добавлено

- 🎨 **Группировка настроек** — логические секции в формах добавления/редактирования счетчика (MQTT, Тарифы, Параметры, Экспорт)
- 💡 **Всплывающие подсказки** — пояснения для каждого параметра
- 🔍 **Предпросмотр MQTT топиков** — сразу видно, какой топик будет использоваться
- ✅ **Улучшенная валидация** — проверка формата времени (ЧЧ:ММ), пороговых значений, портов MQTT (1-65535)

### Исправлено

- 🔧 Асинхронная остановка опроса при выключении HA
- 🔧 Корректное восстановление после перезагрузки
- 🔧 Исправлены обращения к защищенным атрибутам
- 🔧 Исправлена ошибка перевода `MISSING_VALUE`

## [1.3.1] - 2024-01-05

### Добавлено

- Первая публичная версия
- Поддержка двухтарифных счетчиков электроэнергии
- Поддержка счетчиков воды, газа, тепла
- Экспорт показаний в отдельные MQTT топики
- Поддержка отдельного MQTT брокера для экспорта
- Автоматический расчет стоимости по тарифам
- Сохранение состояния при перезагрузке
- Сенсор "Получено импульсов" (имп/мин)
- Ссылка на официальные тарифы в настройках
